### Description (kısa, çünkü TR pazar birincil)

```
For Turkish lawyers (avukat) registered with the bar.

Law fetches your case and hearing data from the UYAP Avukat Portal,
syncs it to your phone calendar, and sends reminders so you never miss
a court date.

PRIVACY FIRST
• Your e-Devlet password never reaches us. You enter it directly into
  the official turkiye.gov.tr site inside the app.
• No backend. All data stays in an AES-256 encrypted database on
  your device.
• No analytics tracking, no data selling.

NOTE
Law is an independent helper app, not an official product of the
Turkish Ministry of Justice. The authoritative source remains the
UYAP Avukat Portal.

Support: [support@DOMAIN]
```

## App Review Notes (Apple inceleme ekibi için)

```
This app helps Turkish lawyers (avukat) view their hearing schedule from
the UYAP Avukat Portal (a government attorney portal). The user logs in
to the official Turkish Government identity gateway (turkiye.gov.tr) via
an in-app WebView, exactly as they would in Safari. The app then reads
the data already visible in the user's authenticated session and stores
it locally on the device (no backend).

REVIEWER TEST CREDENTIALS:
We cannot provide UYAP credentials because UYAP login is tied to a
Turkish citizen ID and is restricted to bar-registered attorneys. We can
provide a screen recording demonstrating the full flow upon request.

The app does not transmit user credentials to our servers. Verification:
the app has no backend; the binary contains no networking calls outside
the WebView and direct HTTPS calls to *.uyap.gov.tr. We are happy to
walk through the code with the review team.

Privacy policy: https://[DOMAIN]/gizlilik
```
